# MealBuddy-Food-Delivery-Management-System
MealBuddy is a web-based food delivery management system designed to simplify online food ordering. The platform enables users to create accounts, browse available food items, add products to their cart, and place orders seamlessly. The system provides a user-friendly interface with secure authentication and efficient order management features.
Features
User Registration and Login
Browse Food Menu
Add to Cart Functionality
Order Placement and Management
User Authentication
Responsive User Interface
Customer Dashboard
Order Tracking and History
Technologies Used
Python
Django
HTML
CSS
JavaScript
SQLite/MySQL (whichever you used)
Objective

The objective of this project is to provide a convenient and efficient online food ordering platform that connects customers with restaurants while simplifying order management.
